
Установка

1)Скопировать в "/local/modules/" и распаковать

2) Установить через Маркетплейс -> Установленные решения

3) Почистить кэш Настройки -> Нстройки продукта -> Автокеширование -> Очистка файлов кеша

4) Если галки в спске не исчезли - Берем бубен:
	Через Контент -> Структура сайта -> Файлы и папки заходим в папку "/local/components/bitrix/crm.timeline/templates/.default/"
	и открываем файл script.js как текст. Сохраняем. Обновляем страницу - галки исчезли - магия Битрикс :)
